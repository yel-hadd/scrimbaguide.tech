# How to Learn React in 2026

<DisclosureNotice />

Solid **JavaScript first** → **components + JSX** → **hooks** (`useState`, `useEffect`, context) → **data fetching** → **router** → **tests + deployment** while building **2 portfolio apps**. Timeline: **~3–6 months** part-time after JS basics.

## Data points to set expectations (US context)

- **Web developer demand is still growing:** the U.S. Bureau of Labor Statistics projects **16% growth** for web developers and digital designers from 2023 to 2033, much faster than average. Source: [BLS Occupational Outlook Handbook](https://www.bls.gov/ooh/computer-and-information-technology/web-developers.htm).
- **JavaScript remains foundational:** Recent Stack Overflow survey data still lists JavaScript as the most commonly used language among professional developers. Source: [Stack Overflow Developer Survey](https://survey.stackoverflow.co/2025/technology#most-popular-technologies-language-prof).
- **Plan for consistency, not intensity:** BLS reports software developer median pay above six figures in the U.S., but the market rewards shipped skills and portfolio depth more than certificate volume. Source: [BLS Software Developers](https://www.bls.gov/ooh/computer-and-information-technology/software-developers.htm).

## 0. Prereq check

If closures and array methods feel foreign, spend 2–4 weeks on [Learn JavaScript](/docs/courses/javascript/) before React.

## 1. Components & JSX

- Function components, props, lists/keys
- Styling approach (CSS modules or Tailwind, pick one)

**Mini project:** static profile page with reusable components.

## 2. Hooks-first React

- `useState`, `useEffect`, custom hooks
- Forms and controlled inputs

**Mini project:** habit tracker or todo with persistence (`localStorage`).

## 3. Data & side effects

- `fetch`, error/loading states
- Optional: TanStack Query once patterns click

**Project:** dashboard hitting a public API.

## 4. Routing & app shape

- React Router (or framework router if you jump to Next.js later)

**Project:** multi-page marketing + app shell.

## 5. Ship it

- Deploy to Vercel/Netlify
- Write README + demo GIF
- Add basic tests (Vitest + RTL) when you repeat patterns
