# Scrimba Roadmap: What's Coming in 2026

<DisclosureNotice />

Still confirmed for 2026: Learn Next.js Section 3, Learn DevOps, and Advanced TypeScript. Unit Testing and Build AI Agents have already shipped. Platform expanding to full-stack and AI.

## Who This Is For

Scrimba users tracking upcoming courses and features, or anyone evaluating whether to subscribe now vs wait for specific content.

Scrimba is evolving rapidly from a frontend-focused platform to a comprehensive full-stack and AI engineering school. Keeping up with their release schedule can be tricky, so we've compiled everything that is officially confirmed for 2026.

## Coming Soon (Confirmed)

According to Scrimba's official course roadmap, the following courses are currently in development or production:

### 1. Learn Next.js (Section 3)
**Status: In Progress**
The "Next.js Developer's Guide" is being expanded. Section 3 will likely cover advanced patterns, streaming, and server actions in greater depth, rounding out what is already Scrimba's most modern framework course.

### 2. Learn DevOps
**Status: Coming Soon**
Bridging the gap between code and production. Expect coverage of CI/CD pipelines, Docker, and perhaps some cloud infrastructure basics (AWS/DigitalOcean).

### 3. Advanced TypeScript
**Status: Coming Soon**
For those who have mastered the basics. This course will dive into generics, utility types (`Pick`, `Omit`, `Partial`), and advanced type inference.

## Recently Added (2025-2026)

If you haven't checked Scrimba in a while, you might have missed these major releases:

- **<AffiliateLink href="https://scrimba.com/learn-cybersecurity-c0ggmpl7f9">Learn Cybersecurity</AffiliateLink>**: A standalone 5-hour course on auth, input safety, rate limiting, and the security mindset.
- **Build AI Agents**: Bob Ziroll's hands-on course on autonomous agents that take actions, use tools, and loop until a goal is achieved (shipped from the roadmap).
- **Unit Testing**: A focused course on testing your code as you write it, covering Vitest/Jest-style patterns (shipped from the roadmap).
- **[Intro to AI Engineering](/docs/courses/ai/ai-engineering/)**: Updated for 2026 with the latest models.
- **[The Backend Developer Path](/docs/paths/backend-developer-path/)**: A massive new dedicated career path.
- **[Learn Context Engineering](/docs/courses/ai/context-engineering/)**: How to optimize context windows for LLMs.
- **Intro to Model Context Protocol (MCP)**: Connecting AI to your local files and data.
- **Authentication & Identity**: Deep dive into auth for the Backend path.

## The Shift to "Fullstack + AI"

Looking at the roadmap, Scrimba's strategy for 2026 is clear: **Move beyond just Frontend.**

1.  **Backend Dominance**: With the Backend Path, recently released Learn Cybersecurity, and upcoming DevOps, Scrimba is positioning itself as a place to learn the *entire* stack.
2.  **AI Integration**: They are doubling down on AI Engineering, not just as a novelty, but as a core engineering discipline.
3.  **Professional Polish**: Courses on Testing and TypeScript signal a focus on "job-readiness" rather than just "project-building."
