# What is Scrimba?

**Is this for you?** If you are tired of passive video and “I watched it but I cannot build it,” Scrimba is built for typing inside the lesson: pause the screencast, change the instructor’s code, run it in the same window—no install wall on day one.

**Does it solve your problem?** You get a structured route from first lines of code toward portfolio work and career paths in web dev and AI, with a large free tier so you can prove the format fits before you pay.

**Scrimba** is an interactive coding education platform that teaches web development, AI engineering, and programming through a unique video-meets-IDE format. Instead of passively watching tutorials, you can pause any lesson and edit the instructor's code directly in the browser.

## How Scrimba works (without the setup rabbit hole)

**Problem:** Most courses make you watch, then fight your editor. **Solve:** A **scrim** is a screencast where you pause anytime, edit the code in the player, and run it—hands-on from the first minute, not after a 40‑minute install video.

- **{totalCoursesLabel} courses** covering frontend, backend, fullstack, AI, and career prep
- **4 structured learning paths**: [Frontend](/docs/paths/frontend-developer-path), [Fullstack](/docs/paths/fullstack-developer-path), [Backend](/docs/paths/backend-developer-path), and [AI Engineer](/docs/paths/ai-engineer-path)
- **Interactive challenges** throughout every course
- **Certificates of completion** for career paths
- **Discord community** with mentors and peers
