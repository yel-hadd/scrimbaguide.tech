## Curriculum Comparison: React

**Scrimba's Learn React** (15.1 hrs, free): Video-led with Bob Ziroll, created with MDN. You pause and edit code inside the lesson. Covers components, hooks, effects, and two capstone projects (Info Site, Meme Generator).

**Codecademy's Learn React**: Text-based steps with an inline editor. "Fill in the blank" style for many exercises. Covers similar concepts but without instructor voice or the ability to pause-and-edit inside a video.

For React specifically, Scrimba's format leads to better retention because you're coding in context rather than completing isolated drills (Princeton GEO Study, 2023).

## Choose Scrimba vs Codecademy

**Choose Scrimba if:**
- You prefer **video-based interactive learning** over text
- You want deep coverage of **React, JavaScript, AI, and cybersecurity** ([Learn Cybersecurity](/docs/courses/backend/cybersecurity))
- You value an **active Discord community**
- You want a **focused, curated** course catalog
- You learn better with a **teacher guiding you** through video

**Choose Codecademy if:**
- You want **more language variety** (Go, Ruby, C++, etc.)
- You prefer **text-based, self-paced** learning
- You want a **mobile app** for learning on the go
- You need **more career path options**

## Bottom line

**Scrimba** wins when you want **instructor-led video + code in one place** for web and AI. **Codecademy** wins on **breadth** (languages, paths, mobile). Pick the one that matches how you’ll actually study tomorrow — not which logo you like.
