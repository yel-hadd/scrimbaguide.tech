# Scrimba Platform Issues & Troubleshooting

**Is this for you?** Something **won’t play**, the **editor glitches**, or **progress looks wrong** — you want fixes, not generic advice.

**Quick answer:** Use **Chrome on desktop** for the smoothest experience. **Refresh**, **clear cache**, or **disable extensions** when the player misbehaves. **Progress oddities** → **help@scrimba.com**. Sections below mirror common Help Centre topics.

*Last reviewed: 2026-03-30.*
