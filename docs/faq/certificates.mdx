## Employer Perspective on Scrimba Certificates

Hiring managers typically evaluate coding bootcamp and platform certificates based on three factors:

1. **Curriculum quality** — Scrimba's Frontend Developer Path was built with Mozilla MDN, a globally recognized authority. This gives it more weight than certificates from unknown platforms.
2. **Practical skills demonstrated** — The career paths include solo projects that become portfolio pieces. These projects, not the certificate itself, are what employers actually assess.
3. **Verification** — Scrimba certificates include a shareable URL. If an employer questions validity, they can verify the certificate directly.

The strongest approach is to list the certificate on LinkedIn under Licenses & Certifications, then link to your portfolio projects in the same profile. The certificate opens the door; the projects close it.
