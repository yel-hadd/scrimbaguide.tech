# Is Scrimba free?

<DisclosureNotice />

**Partially.** Scrimba offers a **Free** tier with **~{freeCount} full courses** (including popular picks like Learn React and Learn JavaScript) and **up to 10 interactive challenges**. **Career paths**, **private Discord**, **unlimited challenges**, and **path certificates** require **Scrimba Pro**. Free courses do not expire as long as Scrimba offers them, verify current listings on Scrimba.

## What stays free vs Pro-only

| | Free | Pro |
|---|------|-----|
| Full courses | ~{freeCount}+ | All {totalCoursesLabel} |
| Interactive challenges | 10 total | Unlimited |
| Career paths | No | Yes (4 paths) |
| Discord | No | Private Pro server |
| Certificates (paths) | No | Yes |
