## About This Course

Learn to build your first web pages in about four hours with HTML and CSS. Kevin Powell, one of the most popular CSS educators online, teaches inside Scrimba’s interactive player so you edit real markup and styles in each lesson.

- **Duration:** 4 hrs
- **Level:** Beginner
- **Access:** Free (no subscription required)

## What Makes This Course Distinctive

Kevin Powell focuses on how HTML and CSS actually fit together for real pages, not abstract lists of tags. The format is the same interactive Scrimba player: you pause, edit the instructor's files, and see results immediately. At four hours it is tighter than the 5.7-hour multi-project HTML and CSS course, so it suits learners who want a single strong primer before JavaScript or a framework. You still get structured progression and practice inside the lesson rather than copying code into a separate editor you might abandon. Pair it with [HTML and CSS](/docs/courses/css/html-and-css) if you want more projects afterward, or move on to Flexbox and Grid once you can build a simple static layout confidently.

## Who Is This Course For?

- **Best for:** Complete beginners and career changers who want a credible, efficient first course in HTML and CSS.
- **Not ideal if:** You already completed a long HTML and CSS track and only need layout systems or Tailwind next.
