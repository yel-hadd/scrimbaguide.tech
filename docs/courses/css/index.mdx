# Scrimba CSS & Design Courses

Scrimba's CSS track covers layout, responsive design, utility frameworks, animation, and visual UI design. CSS is visual, so the interactive format fits it well: you change a value in the lesson and watch the box move.

## Where to start

Start with the free [Learn HTML and CSS](/docs/courses/css/html-and-css) (5.7 hrs). It builds the foundation every other course here assumes. From there the layout pillars, [Learn Flexbox](/docs/courses/css/flexbox) and [Learn CSS Grid](/docs/courses/css/css-grid), carry most modern layouts.

## How to progress

Run [Learn HTML and CSS](/docs/courses/css/html-and-css), then the layout pillars [Learn Flexbox](/docs/courses/css/flexbox) and [Learn CSS Grid](/docs/courses/css/css-grid). Follow with [Learn Responsive Web Design](/docs/courses/css/responsive-web-design), the deepest course here, and [Learn Tailwind CSS](/docs/courses/css/tailwind-css) since Tailwind is the default for new React and Next.js work. If you want the designer's eye and not just the syntax, finish with [Learn UI Design](/docs/courses/css/ui-design).
