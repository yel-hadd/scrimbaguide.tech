## About This Course

This massive UI design course teaches you to build beautiful and well-functioning user interfaces using HTML and CSS. It's filled with interactive tutorials to ensure that the knowledge sticks with you. Created by Gary Simon, the most popular online instructor on this subject.

This Intermediate-level course covers 8.6 hrs of content across 45 interactive screencasts. A [Scrimba Pro](/docs/pricing/) subscription is required for full access.

- **Duration:** 8.6 hrs
- **Level:** Intermediate
- **Access:** [Scrimba Pro](/docs/pricing/) required
- **Modules:** 12
- **Total Lessons:** 45

## What Makes This Course Distinctive

This is Scrimba's full UI design program with Gary Simon: multiple projects, responsive navigation work, refactoring practice, and deep dives into shadows, gradients, forms, and UI animation before a substantial final project. Everything stays in HTML and CSS, which keeps deliverables aligned with what frontend interviews actually ask you to produce. Interactive lessons mean you adjust real markup and styles instead of only admiring finished mockups. The first module overlaps with [UI Design Fundamentals](/docs/courses/css/ui-design-fundamentals) in spirit, but this path adds breadth and sustained project work. Choose fundamentals if you need a quick Pro taste first. Skip this track if you refuse long coursework or only want utility-framework styling.
