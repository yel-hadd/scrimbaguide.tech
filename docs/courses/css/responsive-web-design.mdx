## Why this course (problem → outcome)

Desktop layouts collapse on mobile, or you stack random media queries without a system, slow on the job and weak in interviews.

**Outcome:** Six Pro modules rebuild fundamentals, then **responsive thinking**, styling depth, Flexbox, Grid, and harder project-style work, with **challenges** so you practice in context. Scrimba’s interactive format means you change CSS in the lesson and see the result immediately, not passive video alone.
