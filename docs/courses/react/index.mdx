# Best Scrimba React courses (2026)

**For you if** you want **one map** of Scrimba’s React lineup,**free** [Learn React](/docs/courses/react/learn-react) (~15 hrs, Bob Ziroll) first, then **Pro** adds Advanced React, Router, challenges, games, Styled Components, interview prep, and more. **Solves** “what order?” and “where does free end?” without opening twelve marketing pages.

Scrimba lists **12 React-focused courses**, a deep catalog in **one interactive player** (you type in the scrim, not only watch). That consistency matters next to scattered YouTube playlists.

## Why use this index

Learners skip fundamentals and stall on Advanced React, or buy Pro without realizing **Router** is its own course.

We show **hours**, **level**, and **Free vs Pro** in one table and a **recommended order** so you pick a track, then open individual reviews for prerequisites and projects.

## Who is this for?

Career changers and beginners who want a single source of truth for Scrimba React options, plus intermediate devs picking the next Pro module. Helpful if you follow the Frontend or Fullstack paths and want to see how React courses line up. Less useful if you already finished Learn React and only need one specific deep dive (open that course's page directly).
