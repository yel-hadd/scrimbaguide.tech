# Scrimba React Courses

Scrimba's React track runs from a free, project-heavy flagship all the way to advanced patterns, routing, and interview prep. Every course is interactive: you pause the lesson and edit the code in the same browser tab.

## Where to start

Start with the free [Learn React](/docs/courses/react/learn-react) (15.1 hrs). It covers components, props, state, hooks, and finishes on two portfolio projects. It is the anchor for everything else in this category, so do it first whether or not you ever pay for Pro.

## A sensible order

Run [Learn React](/docs/courses/react/learn-react) first, then add [Advanced React](/docs/courses/react/advanced-react) for production patterns and [Learn React Router](/docs/courses/react/react-router) for navigation. Once those land, use [React Challenges](/docs/courses/react/react-challenges) for reps and [React Interview Questions](/docs/courses/react/react-interview-questions) when interviews are near. [What's New in React 19?](/docs/courses/react/whats-new-in-react-19) is a short update tour for people already comfortable with React 18.

## Where it fits

React sits inside two career tracks on Scrimba Pro. The [Frontend Developer Path](/docs/paths/frontend-developer-path) wraps Learn React and Advanced React into a longer HTML, CSS, JS, and TypeScript sequence. The [Fullstack Developer Path](/docs/paths/fullstack-developer-path) adds Node, Express, and Next.js after React. Browse every track on the [Learning Paths hub](/docs/paths/).
