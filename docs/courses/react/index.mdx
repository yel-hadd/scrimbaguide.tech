# Scrimba React Courses

*Last reviewed: May 2026. We may earn a commission if you join Scrimba through links on this page; the order and the "skip if" notes are independent.*

**Quick answer.** If you are starting from zero, run free **Learn React** (Bob Ziroll, 15.1 hrs) first, then add **Advanced React** and **React Router** on Pro. Everything else is a side quest. The catalog is strong on fundamentals and weak on framework-meta topics like server components and Vite/Webpack tooling. We will say so where it matters.

Scrimba currently ships 9 React-tagged courses. Counts and durations on this page come straight from our [`data/courses.json`](https://github.com/yel-hadd/scrimbaguide.tech/blob/main/data/courses.json) snapshot, not from upstream marketing copy.

## Who this page is for

You are picking your next React module on Scrimba and you do not want to open twelve marketing tabs. You want one ordered map, the honest skip notes, and a clear view of what Scrimba does not cover.

If you have already finished Learn React and you only need one specific deep dive, jump straight to that course page in the table below instead.

The path most learners do best with is short. Three Pro courses plus one free intro will cover what teams ship in 2026 React codebases.

## Honest "skip this course if" notes for the side quests

**React Challenges.** Skip if you have not finished Learn React. The challenges assume hooks fluency and will frustrate you otherwise. If you are post-Learn-React and want reps, this is the strongest practice asset in the catalog.

**Build a Memory Game.** Skip if you are not interested in accessibility patterns. The real value here is keyboard navigation and ARIA, not the game itself.

**Reusable React Components.** Skip if you have not yet shipped a component twice. The patterns (compound components, slots, render props) only click once you have felt the pain they solve.

**Styled Components.** Skip if your stack uses Tailwind or CSS Modules. Styled Components is on the way out for new projects; learn it only if you are joining an existing codebase that uses it.

**Class Components.** Skip unless you are maintaining a pre-2019 React codebase. Bob is honest in the description that this is legacy. New code should not use class components.

**Movie Search App / Tic Tac Toe.** Skip if Learn React's capstones already gave you the build-something-small reps. These are fine warm-ups but they overlap heavily with the flagship.

**React Interview Questions.** Skip if you want behavioral or system-design interview prep. This is a 40-minute concept refresh, not mock-interview practice. Pair it with our [React interview questions guide](/docs/courses/react/react-interview-questions) if you want depth.

## Bob Ziroll's teaching style, briefly

We tried to pull a representative passage from Bob's Scrimba React intro through yt-dlp on May 23, 2026. YouTube returned 429s on subtitle endpoints and DRM errors on the player API, so we have no fresh transcript to quote. Rather than fabricate one, we describe the style from verifiable public materials.

Bob teaches in two registers. In Learn React, he walks slowly, asks you to pause and write code before he writes it, and admits when a concept is awkward (his takes on `useEffect` cleanup and Strict Mode double-invocation are unusually candid). In Advanced React, the pace tightens. The Scrimba teacher page documents his work and freeCodeCamp hosts hours where the same cadence is audible ([Scrimba teacher page](https://scrimba.com/teachers/bob-ziroll), [freeCodeCamp author page](https://www.freecodecamp.org/news/author/bobziroll/)).

## Where Scrimba's React catalog is thin

We will not pretend the catalog is complete. Three real gaps as of May 2026:

1. **No deep server components or Next.js App Router course taught as React.** Learn Next.js exists (4.4 hrs) but it is intentionally light. Bridge: Lee Robinson's [Next.js tutorials on YouTube](https://www.youtube.com/@leerob) and the [official Next.js Learn course](https://nextjs.org/learn) cover Server Components and the App Router at the depth Scrimba does not.
2. **No build-tooling course.** There is a short Intro to Vite (39 min) tagged under JavaScript, but no module on Webpack, esbuild, or modern Vite plugin authoring. Bridge: the [Vite guide](https://vitejs.dev/guide/) plus Theo Browne's [t3.gg YouTube channel](https://www.youtube.com/@t3dotgg) for working-dev tooling content.
3. **No state-management deep dive.** No Zustand, Redux Toolkit, Jotai, or TanStack Query course. Bridge: [TanStack Query docs](https://tanstack.com/query) for server state, [Zustand docs](https://zustand.docs.pmnd.rs) for client state. Both are 30-minute reads.

If these gaps are blockers for your role, pair Scrimba with the [official React docs](https://react.dev). Scrimba covers fundamentals and patterns; the framework-meta layer lives elsewhere.

For a third-party take on the flagship, see Bob Ziroll's [Learn React on freeCodeCamp's YouTube](https://www.youtube.com/watch?v=bMknfKXIFA8) (the public version of the Scrimba intro) or the dev.to community reviews under the [#scrimba tag](https://dev.to/t/scrimba).
