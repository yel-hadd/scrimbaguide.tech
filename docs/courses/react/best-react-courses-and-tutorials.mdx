# Best React Courses and Tutorials on Scrimba, Compared

Scrimba runs a full React track, from a free flagship course to advanced patterns, routing, project builds, and interview prep. They all use the same interactive format: you pause the video and edit the instructor's code in the same browser tab. This page compares the options side by side so you can pick the right one for where you are, rather than starting them in the wrong order.

## Which one should you pick?

- **Brand new to React?** Start with <AffiliateLink href="https://scrimba.com/learn-react-c0e">Learn React</AffiliateLink>. It is free, it is the longest course in the track, and it ends on real projects. It anchors everything else, so do it first whether or not you ever pay for Pro.
- **Comfortable with components and hooks?** Move to [Advanced React](/docs/courses/react/advanced-react) for the patterns you hit in real codebases, then add [Learn React Router](/docs/courses/react/react-router) once you need more than one page.
- **Learn best by doing, not watching?** [React Challenges](/docs/courses/react/react-challenges) is the most active-practice course in the set. Pair it with a project build like [Build a Memory Game in React](/docs/courses/react/memory-game-in-react).
- **Interviewing soon?** [React Interview Questions](/docs/courses/react/react-interview-questions) is a short, targeted refresher rather than a full course.
- **Maintaining a legacy app?** [Learn Class Components in React](/docs/courses/react/class-components-in-react) covers the older syntax you will still meet in existing codebases.

If you want React inside a structured, job-focused sequence instead of picking courses one by one, the React track is built into the <AffiliateLink href="https://scrimba.com/frontend-path-c0j">Frontend Developer Path</AffiliateLink> (81.6 hrs) and the <AffiliateLink href="https://scrimba.com/fullstack-path-c0fullstack">Fullstack Developer Path</AffiliateLink> (108.4 hrs).

## Prerequisites

You need a working grasp of HTML, CSS, and JavaScript, especially arrays, functions, and ES6 syntax, before the React courses will click. If any of that feels shaky, start with [Learn JavaScript](/docs/courses/javascript/learn-javascript) first.
