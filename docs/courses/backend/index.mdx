# Scrimba Backend Courses

Scrimba's backend courses cover databases, backend-as-a-service, and the building blocks behind real APIs. The stack is JavaScript and TypeScript throughout, so it fits frontend developers who want to add persistence, auth, and data without leaving the language they know.

## Where to start

Start with the free [Learn SQL](/docs/courses/backend/sql) (3.8 hrs). Database literacy is what separates copy-paste developers from people who can debug a slow query, and it pays off whether you later use raw SQL, Supabase, or Firebase.

## All backend courses

## How to progress

Run [Learn SQL](/docs/courses/backend/sql) for database fundamentals, then [Intro to Supabase](/docs/courses/backend/supabase) for SQL plus auth, storage, and row-level security in one platform. [Build a Mobile App with Firebase](/docs/courses/backend/a-mobile-app-with-firebase) covers Google's backend-as-a-service if you want easy mobile sync instead.

## More backend on the JavaScript track

Scrimba files its server-side runtimes under JavaScript. For the request-response side of backend work, follow this hub with [Learn Node.js](/docs/courses/javascript/nodejs), [Learn Express.js](/docs/courses/javascript/expressjs), and [Intro to NestJS](/docs/courses/javascript/nestjs) on the [JavaScript hub](/docs/courses/javascript/).

## Where it fits

The fastest structured route is the [Backend Developer Path](/docs/paths/backend-developer-path), which sequences Node, Express, SQL, TypeScript, and deployment into one progression. The broader [Fullstack Developer Path](/docs/paths/fullstack-developer-path) adds the frontend half. Browse every track on the [Learning Paths hub](/docs/paths/).
