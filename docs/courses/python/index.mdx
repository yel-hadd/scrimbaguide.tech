# Scrimba Python Courses

**Quick answer:** Scrimba has one Python course, [Learn Python](/docs/courses/python/learn-python), at roughly 5.6 hours and 74 lessons on the Pro tier. It covers variables, control flow, data structures, functions, and small projects inside the browser. It is a frictionless way to write your first Python without installing a toolchain. It is not a substitute for a Python-first program if your real goal is Django, FastAPI, data science, or machine learning. Use this hub to see how the course fits, then plan where to go after it.

## The problem most beginners hit

You open a Python tutorial, spend an hour wrestling with installs and `PATH`, then watch someone type while your attention drifts. You never get reps writing logic yourself, so syntax never sticks.

## How Scrimba handles that

Scrims are interactive screencasts. You pause the instructor, edit the same code in the same window, and run it. That is closer to real practice than passive video, and you do it without fighting your laptop first.

## Honest scope: what this course is and is not

It is a confident, beginner-friendly intro to Python syntax and core patterns. It is not a full data science track, a Django bootcamp, or an interview-prep grinder. Scrimba's catalog leans JavaScript, React, and AI, so Python here is one well-built course, not a multi-path specialisation. Plan to supplement with a book or a focused program once your goal narrows.

## Bridges for what Scrimba does not cover

No multi-course Python track on Scrimba. Where to go after Learn Python, by goal:

- **Web (Django, Flask, FastAPI).** Bridges: [Django official tutorial](https://docs.djangoproject.com/en/stable/intro/tutorial01/), [FastAPI tutorial](https://fastapi.tiangolo.com/tutorial/), and Miguel Grinberg's [Flask Mega-Tutorial](https://blog.miguelgrinberg.com/post/the-flask-mega-tutorial-part-i-hello-world).
- **Data and ML.** Bridges: [fast.ai](https://www.fast.ai/) for practical deep learning, [Andrew Ng's Machine Learning Specialization](https://www.coursera.org/specializations/machine-learning-introduction) for the canonical on-ramp.
- **Real-world Python idioms.** Bridge: [Real Python's free tutorials](https://realpython.com/) and the [official Python tutorial](https://docs.python.org/3/tutorial/).
- **Interview prep.** Bridge: [Exercism's Python track](https://exercism.org/tracks/python) for short reps with mentor feedback.

## What you will learn

- Variables and types: strings, numbers, booleans, coercion
- Control flow: `if`, `elif`, `else`, `for`, `while`
- Data structures: lists, dicts, tuples, string methods
- Functions: define, call, return, scope
- Files: read and write text
- Common patterns: sorting, searching, simple algorithms
- Interactive challenges throughout

## Python vs JavaScript (if you came from the JS track)

| Concept | Python | JavaScript |
|---------|--------|------------|
| Blocks | Indentation | `{ }` |
| Variables | Assign directly | `let`, `const` |
| Falsy values | `None`, `""`, `0`, `[]` | `null`, `""`, `0`, `[]` |
| Collections | `list`, `dict` | `Array`, `Object` |
| Imports | `import` from PyPI | `import` from npm |

Python's [standard library](https://docs.python.org/3/library/) covers JSON, HTTP, files, and regex, which is useful before you reach for a package for everything.

## Where Python can take you after the intro

- Backends with Django, FastAPI, or Flask
- Data and ML with Pandas, NumPy, scikit-learn
- Automation: scripts, scraping, scheduled jobs
- APIs: REST and GraphQL services
