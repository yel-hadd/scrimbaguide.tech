## About This Course

Learn to build web apps with Next.js, the world's most popular full-stack framework. You'll cover routing, layouts, data fetching, optimizing assets, and more, while building PrintForge, a site for 3D printing enthusiasts.

This Intermediate-level course covers 4.4 hrs of content across 32 interactive screencasts. A [Scrimba Pro](/docs/pricing/) subscription is required for full access.

- **Duration:** 4.4 hrs
- **Level:** Intermediate
- **Access:** [Scrimba Pro](/docs/pricing/) required
- **Modules:** 2
- **Total Lessons:** 32

## What Makes This Course Distinctive

The course anchors everything in one product-shaped app, PrintForge, so routing and data fetching decisions feel motivated by user flows, not toy examples. The first half builds the Next.js surface area you touch daily: pages or app patterns, layouts, and how data gets to the UI. The second half pushes into rendering choices and performance-related ideas so you understand why Next exists beyond "React with folders." Interactive screencasts mean you edit the same project the instructor uses, which reduces the usual gap between docs and a broken local setup. It is honest intermediate material: you should already speak React. Graduates get a clearer map of fullstack React than from scattered blog posts. It will not replace official docs for every edge case or the newest release notes.

React developers moving toward fullstack roles, freelancers who need SSR and routing without assembling everything by hand, and path learners on the [Frontend Developer Path](/docs/paths/frontend-developer-path) or [Fullstack Developer Path](/docs/paths/fullstack-developer-path) who want Next as the capstone framework.
