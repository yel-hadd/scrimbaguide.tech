## About This Course

In this course, you’ll learn two different ways to deploy your projects to the web with Netlify. We’ll cover how to drag and drop deploy a project in seconds, as well as how to automatically deploy changes to your codebase with continuous deployment.

This Intermediate-level course covers 23 min of content. It is available for free — no subscription required.

- **Duration:** 23 min
- **Level:** Intermediate
- **Access:** Free (no subscription required)

## What Makes This Course Distinctive

The course is short because deploying a well-built static site should be short. You get two workflows: instant drag-and-drop for prototypes, and Git-linked continuous deployment for anything you iterate on. Interactive screencasts mean you click the same Netlify UI the instructor uses, which reduces the usual mismatch between docs screenshots and your dashboard version. You will not learn Kubernetes or custom servers here; that is the point. It pairs cleanly with portfolio and frontend projects elsewhere on Scrimba when you need a credible URL for employers. If your build pipeline is already complex, you may still need team-specific CI docs beyond this intro.

## Prerequisites

Basic knowledge of HTML and CSS is recommended before starting this course.

## Who Is This Course For?

Beginners and intermediates who built HTML, CSS, or JS projects locally and need hosting. Career switchers polishing a portfolio also fit.

**Not ideal if:** You need enterprise Netlify teams, edge functions deep dives, or self-hosted deployment only.
