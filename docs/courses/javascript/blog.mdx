## Who this is for
You want official Scrimba context between deep courses, release notes, tips, what to take next.

The blog is **editorial**, not a bounded syllabus. Bookmark it **next to** structured tracks like [Learn JavaScript](/docs/courses/javascript/learn-javascript). Reading is free; interactive courses may need [Pro](/docs/pricing/).

- **Format:** Articles and pointers (open-ended)  
- **Access:** Blog is free to read; linked courses vary. See [Pro vs Free](/docs/pricing/pro-vs-free)  

## Why it’s useful

You get the **official voice** on updates and motivation, plus bridges into longer interactive courses. It’s a weak **sole** resource if you need one project with a clear lesson count, pair it with a path or flagship course.

Basic knowledge of HTML and CSS is recommended before starting this course.

## Who Is This Course For?

Scrimba members who want official articles, motivation, and pointers between courses, plus newcomers orienting before they pick a structured track.

**Not ideal if:** You need a bounded curriculum with modules, homework, and completion certificates as the only goal.

## Choose This If

**Choose this course if:**

- You want the blog hub bookmarked next to interactive courses in your learning routine.
