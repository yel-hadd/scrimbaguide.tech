## About This Course

This ES6+ tutorial walks modern JavaScript in **23 interactive screencasts**: a deliberate order from syntax upgrades to patterns you’ll see in real React and Node code, not a random blog stack.

- **Duration:** 119 min
- **Level:** Intermediate
- **Access:** [Scrimba Pro](/docs/pricing/) required

## What Makes This Course Distinctive

Twenty-three screencasts sounds small until you remember each one expects you to change code, not only watch. The course walks ES6+ features in a deliberate order so you see how destructuring, modules, classes, and newer collection helpers relate to everyday component and utility code. That structure beats jumping between MDN pages when you are trying to standardize how you write.

It sits between a micro syntax refresh and a full advanced course. You get more room to breathe than the 52-minute [Learn Modern JavaScript](/docs/courses/javascript/modern-javascript) variant, but you are still focused on language updates, not building a large app. The interactive format helps when you are unlearning patterns from older tutorials (`var`, long callback chains) and want immediate feedback.

Best payoff: you read open-source JavaScript without getting lost in unfamiliar syntax. Weakest fit: absolute beginners who still struggle with functions and control flow. This course assumes you can already ship small scripts and want a coherent ES6+ baseline.

## Who Is This Course For?

Best for: developers who know the basics but want a full ES6+ baseline: modules, classes, and modern syntax used in real codebases.

Not ideal if: you're brand new to JavaScript. Start with [JavaScript Tutorial](/docs/courses/javascript/learn-javascript).
