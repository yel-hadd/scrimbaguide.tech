## About This Course

Use **LangChain Expression Language (LCEL)** in JavaScript to compose LLM steps, tools, and retrieval into a **chatbot-shaped project** you can reason about—not a one-off prompt in a playground.

- **Duration:** 94 min
- **Level:** Intermediate
- **Access:** [Scrimba Pro](/docs/pricing/) required

## What Makes This Course Distinctive

The course centers on LangChain.js and LCEL, which is how many teams compose prompts, retrievers, and tools without writing spaghetti glue code. Interactive screencasts help because LLM pipelines fail in subtle ways, and you want to tweak one step and rerun immediately. You build toward a chatbot-shaped project so the abstractions connect to user-visible behavior, not notebook snippets alone. Expect a practical slice of the ecosystem rather than every integration and provider. It lines up with the [AI Engineer Path](/docs/paths/ai-engineer-path) if you are stacking AI courses on Scrimba. You still need patience for fast-moving library APIs and should pair this with general prompt and safety reading outside the course.

## Who Is This Course For?

JavaScript developers exploring AI app stacks who want LangChain-style composition with real typing practice. Backend or fullstack engineers adding LLM features to existing products also fit.

**Not ideal if:** You want a Python-only LangChain track, or you need production evals, red-teaming, and compliance as the main curriculum.
