## About This Course

Learn how to build LLM-powered agents that can do multi-step reasoning and interact with its environment via functions and APIs.

- **Duration:** 117 min
- **Level:** Intermediate
- **Access:** [Scrimba Pro](/docs/pricing/) required

## What Makes This Course Distinctive

The course pushes past single-prompt demos into agents that plan steps and call functions when the model needs structured actions. Interactive screencasts matter because agent bugs are often wiring bugs: wrong arguments, missing guards, or bad error handling around API calls. Projects like an AI travel agent give you a narrative thread so the abstraction feels like software design, not a notebook trick. It sits on the [AI Engineer Path](/docs/paths/ai-engineer-path) and pairs with other Scrimba AI courses when you are building a full toolkit. You should expect fast-changing provider APIs and validate anything safety-critical outside class hours. The runtime assumes comfort with async JavaScript and reading stack traces.

## Who Is This Course For?

Fullstack and frontend developers adding agentic features to apps, and engineers who completed an AI intro and want orchestration patterns next.

**Not ideal if:** You want pure ML training, computer vision only, or compliance-heavy production agent deployment as the sole focus.
