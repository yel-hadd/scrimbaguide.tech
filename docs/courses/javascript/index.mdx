# Scrimba JavaScript Courses

JavaScript is the widest category on Scrimba. It spans the core language, the frameworks and runtimes built on top of it, the tooling around it, and career prep. Every course is interactive, so you write code in the lesson rather than just watch.

## Where to start

Start with the free [Learn JavaScript](/docs/courses/javascript/learn-javascript) (9.4 hrs). It teaches the language through four built projects rather than a single capstone, which keeps momentum high. It is the anchor for the rest of this category.

Alongside the courses, Scrimba publishes a free written <AffiliateLink href="https://docs.scrimba.com/javascript/introduction" location="companion-docs">JavaScript reference</AffiliateLink> in its docs. It reads as a textbook to the interactive scrims, with a Beginner, Intermediate, and Deep Dive switcher that adjusts how deep each topic goes, so you can look something up or revisit it at more depth as your JavaScript matures.

## How to progress

Run [Learn JavaScript](/docs/courses/javascript/learn-javascript) first, then add [Advanced JavaScript](/docs/courses/javascript/advanced-javascript) and [The Tricky Parts of JavaScript](/docs/courses/javascript/tricky-parts-of-javascript) for language depth. From there, branch by goal: [Learn Node.js](/docs/courses/javascript/nodejs) and [Learn Next.js](/docs/courses/javascript/nextjs) for the runtime and framework side, or [JavaScript Interview Challenges](/docs/courses/javascript/javascript-interview-challenges) when interviews are near.

## Where it fits

The [Frontend Developer Path](/docs/paths/frontend-developer-path) wraps the core JS courses into a longer track, the [Fullstack Developer Path](/docs/paths/fullstack-developer-path) extends it with Node, Express, and Next.js, and the [Backend Developer Path](/docs/paths/backend-developer-path) focuses on async JS and servers. Browse them all on the [Learning Paths hub](/docs/paths/).
