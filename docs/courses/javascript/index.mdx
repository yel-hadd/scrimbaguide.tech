# Scrimba's JavaScript catalog, mapped without the hype

*Last reviewed: May 2026. ScrimbaGuide may earn a commission when you subscribe through links on this page. The order and the "skip if" notes are independent.*

**Quick answer.** Start free with **Learn JavaScript** (Per Borgen, 9.4 hrs, 4 built projects). Then add **Advanced JavaScript** with Tom Chant on Pro for depth. Pair with **JavaScript Interview Challenges** when interviews are near. The flagships are strong; the gaps are mostly in modern tooling and TypeScript-adjacent topics.

Counts and durations on this page come from our [`data/courses.json`](https://github.com/yel-hadd/scrimbaguide.tech/blob/main/data/courses.json) snapshot.

A note on instructor names. The task brief lists Bob Ziroll on the JavaScript flagship; the live Scrimba course and our catalog snapshot list **Per Borgen** (Scrimba's CEO) as the Learn JavaScript instructor. We list the actual instructor here.

## Who this page is for

You are picking a JavaScript track on Scrimba and want one ordered map rather than a "top 50 JS resources" list. You want the access (Free vs Pro) flagged before you pay and an honest read on which side quests are worth your time.

## ES6+ deep dives, ranked

After Learn JavaScript and before Advanced JavaScript, pick one of these. There is real overlap; doing all three is wasteful.

- **Introduction to ES6+.** Best if you learned JS pre-2018 and want a refresh on `let`/`const`, arrow functions, destructuring, modules, spread.
- **Modern JavaScript.** Best if you want a project-flavored tour over a feature checklist.
- **[The Tricky Parts of JavaScript](/docs/courses/javascript/tricky-parts-of-javascript).** Best if syntax is fine but coercion, `this`, and hoisting still bite.

## Honest "skip this course if" notes for the side quests

**JavaScript Deep Dive.** Skip if you completed Advanced JavaScript. Heavy overlap and Deep Dive is older.

**DSA / Binary Search / Merge Sort.** Skip if you are not interview-prepping. A starting nudge, not a replacement for LeetCode.

**Frontend Interview Tips.** Skip if you only want code questions. Leans behavioral and interview-format.

**Unit Testing / Clean Code.** Skip if you already test at work. Friendly intros for first-team self-taught devs.

**Intro to Vite (39 min).** Skip unless you are setting up a project this week. Intentionally tiny.

**Astro / Svelte / Vue / NestJS.** Skip unless you chose that framework. They are tours, not bootcamps.

**Space Travel Website / Deploy Your Portfolio.** Skip if Learn JavaScript's four projects already gave you enough reps.

## Where Scrimba's JavaScript catalog is thin

Three honest gaps as of May 2026:

1. **No deep TypeScript for JS devs.** Scrimba has one TypeScript course in its own category. Bridge: Matt Pocock's [Total TypeScript free essentials](https://www.totaltypescript.com/tutorials) remains the standard "TS for JS devs" resource on the web.
2. **No modern tooling course.** Intro to Vite is 39 minutes. No Webpack, esbuild, Turbopack, or build-debugging content. Bridge: the [Vite guide](https://vitejs.dev/guide/) and Theo Browne's [t3.gg YouTube channel](https://www.youtube.com/@t3dotgg) cover real tooling debugging.
3. **No async-and-concurrency deep dive.** Advanced JavaScript covers async basics. No full course on Promises, async iterators, AbortController, or streams. Bridge: Jake Archibald's [HTTP 203 talks](https://www.youtube.com/playlist?list=PLNYkxOF6rcICWx0C9LVWWVqvHlYJyqw7g) and his [tasks/microtasks article](https://web.dev/articles/in-the-loop) on web.dev.

If these gaps block your role, pair Scrimba with [MDN](https://developer.mozilla.org/en-US/docs/Web/JavaScript).

For third-party takes on the flagship, see Per Borgen's own [freeCodeCamp posts](https://www.freecodecamp.org/news/author/perborgen/) and community write-ups under the [dev.to #scrimba tag](https://dev.to/t/scrimba).
