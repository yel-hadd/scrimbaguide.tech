# Scrimba TypeScript Courses

Scrimba treats TypeScript as a layer on top of the JavaScript you already write, not a separate language reboot. The catalog is one focused, free course tied to the React and Node stacks teams actually ship.

## Where to start

There is one course here: the free [Learn TypeScript](/docs/courses/typescript/learn-typescript/) (4.2 hrs). It covers type fundamentals, TypeScript in React, and TypeScript in Express, plus a hands-on project. It assumes you already know JavaScript, so do a JS course first if you do not.

## All TypeScript courses

| Course | Level | Access | Length |
|---|---|---|---|
| [Learn TypeScript](/docs/courses/typescript/learn-typescript/) | Intermediate | Free | 4.2 hrs |

## How to progress

The prerequisite is JavaScript: run [Learn JavaScript](/docs/courses/javascript/learn-javascript/) first if you have not, then [Learn TypeScript](/docs/courses/typescript/learn-typescript/) to add types, safer refactors, and the skill most React and Node job posts ask for. After that, apply it in the framework courses on the [React](/docs/courses/react/) and [backend](/docs/courses/backend/) hubs.

## Where it fits

TypeScript modules appear inside the [Frontend Developer Path](/docs/paths/frontend-developer-path/), the [Fullstack Developer Path](/docs/paths/fullstack-developer-path/), and the [Backend Developer Path](/docs/paths/backend-developer-path/), which reflects hiring reality. Browse every track on the [Learning Paths hub](/docs/paths/).
