A new kind of coding course that's less about the details of coding and more about creating, experimenting, and realizing your ideas, even without the coding know-how!

This Beginner-level course covers 4.5 hrs of content across 23 interactive screencasts. A [Scrimba Pro](/docs/pricing/) subscription is required for full access.

## What Makes This Course Distinctive

This is Scrimba’s on-ramp for people who want to create with AI assistants before they worry about every syntax detail. Across three modules you move from first deployable projects with ChatGPT through more ambitious builds and a Firebase-backed project. The point is not to memorize every keyword; it is to learn how to steer an assistant, recover when output is wrong, and finish something you can show.

The interactive format matches how people actually work now: you type, you run, you adjust the prompt. That helps the “vibe coder” persona who already ships experiments but wants the code to stop breaking in opaque ways. It also suits true beginners who need permission to build first and refine second.

AI engineer compensation bands often sit around **$140,000-$180,000+** for people who combine web delivery skills with model fluency. This course does not replace a full API track, but it builds the collaboration habits that make later courses on the [AI Engineer Path](/docs/paths/ai-engineer-path) stick. You practice shipping, not only watching.
