## About This Course

Learn to build AI-powered apps with modern LLM tooling, the fastest-growing layer of today’s software stack. You will cover prompts, tokens, system messages, streaming, and more while building **GiftGenie**, an AI-powered gift idea generator.

This Intermediate-level course covers 2.5 hrs of content. A [Scrimba Pro](/docs/pricing/) subscription is required for full access.

- **Duration:** 2.5 hrs
- **Level:** Intermediate
- **Access:** [Scrimba Pro](/docs/pricing/) required

## What Makes This Course Distinctive

You build GiftGenie, an AI-powered gift idea generator, while learning the same primitives employers expect on AI engineer job posts: prompts, tokens, system messages, and streaming responses. The work happens in JavaScript against real APIs, so frontend and fullstack developers can ship features in the stack they already use. Glassdoor-style ranges for AI engineer roles often land around **$140,000-$180,000+**; this course targets the practical layer (calling models, shaping outputs, wiring UIs) rather than abstract ML theory.

Scrimba’s interactive player matters here because API courses die on setup friction. You run and edit code inside the lesson, see errors immediately, and iterate without configuring a local toolchain first. That helps both the “vibe coder” who leans on AI tools but wants to read the request/response cycle, and the experienced developer who needs a fast path to production-shaped patterns.

The course sits on the [AI Engineer Path](/docs/paths/ai-engineer-path), so it is a deliberate first step before RAG, agents, and provider-specific deep dives. You leave with a named portfolio piece and vocabulary that transfers to interviews and code reviews.
