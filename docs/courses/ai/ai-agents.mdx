## About This Course

Learn how to build LLM-powered agents that can do multi-step reasoning and interact with its environment via functions and APIs.

This Intermediate-level course covers 117 min of content. A [Scrimba Pro](/docs/pricing/) subscription is required for full access.

- **Duration:** 117 min
- **Level:** Intermediate
- **Access:** [Scrimba Pro](/docs/pricing/) required

## What Makes This Course Distinctive

Agents are where LLMs stop sounding like autocomplete and start behaving like small programs: they choose tools, call functions, and chain steps toward a goal. You work toward an AI Travel Agent and explicit action functions so the pattern is visible in code, not hidden inside a hosted assistant UI.

That skill set maps cleanly to what companies pay for in senior AI product engineering. Published salary bands for AI engineer positions often sit around **$140,000-$180,000+**. This course keeps the stack familiar for web developers (JavaScript, APIs) so you can connect agent logic to real backends you already maintain.

Bob Ziroll teaches on Scrimba with the same interactive format that matters for agent work: when a tool call fails or a plan loops, you want to edit one function and replay immediately. Use it as a core module on the [AI Engineer Path](/docs/paths/ai-engineer-path) after you are comfortable with basic model calls.

### Instructor

This course is taught by **Bob Ziroll**, an experienced educator on the Scrimba platform.

## What you'll build

- **Tool-calling patterns** (“action functions”) you can explain in code review.
- **AI Travel Agent**, multi-step agent behavior with APIs and decisions you can show in a portfolio.
