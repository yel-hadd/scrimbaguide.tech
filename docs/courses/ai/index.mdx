# Scrimba AI Courses

Scrimba's AI track teaches LLM application development in JavaScript and TypeScript: prompt engineering, agents, RAG, MCP, and the Vercel AI SDK. It is aimed at web developers who want to ship AI features, not at Python ML researchers.

## Where to start

Start with the free [Learn to Code with AI](/docs/courses/ai/to-code-with-ai) (4.5 hrs) or the free [Intro to Mistral AI](/docs/courses/ai/mistral-ai) (1.4 hrs). Both let you touch the workflow without a Pro subscription before you commit to the deeper engineering courses.

## How to progress

Begin with the free [Learn to Code with AI](/docs/courses/ai/to-code-with-ai), then add [Intro to AI Engineering](/docs/courses/ai/ai-engineering) for the JS-dev-meets-LLMs foundation. Layer on [Learn AI Agents](/docs/courses/ai/ai-agents) and [Learn RAG](/docs/courses/ai/rag) for the two core application patterns, then [Intro to Model Context Protocol (MCP)](/docs/courses/ai/model-context-protocol-mcp) and [Build a Support Agent with Vercel AI SDK](/docs/courses/ai/a-support-agent-with-vercel-ai-sdk) for the current plumbing.

## Where it fits

These courses are sequenced into the [AI Engineer Path](/docs/paths/ai-engineer-path) on Scrimba Pro, the single best reason to subscribe in this category. Browse every track on the [Learning Paths hub](/docs/paths/).
