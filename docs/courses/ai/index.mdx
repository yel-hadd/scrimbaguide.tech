## The 2026 AI stack (and where Scrimba sits)

A lot changed since the early agent hype:

- **MCP is no longer experimental.** Anthropic donated the Model Context Protocol to the Linux Foundation's Agentic AI Foundation in December 2025. OpenAI, Microsoft, Google, AWS, and Cloudflare all ship MCP clients or servers. Public servers now number in the thousands. If you are building agents in 2026 and you have not touched MCP, you are behind.
- **Vercel AI SDK has matured** into the default for streaming UIs in Next.js and React. Tom Chant's [support agent build](/docs/courses/ai/a-support-agent-with-vercel-ai-sdk) walks the exact pattern most product teams ship.
- **LangChain.js still has critics.** It is useful when you need glue across vendors, frustrating when its abstractions hide what is actually happening on the wire. Scrimba's [LangChain.js course](/docs/courses/ai/langchainjs) teaches it; treat the abstractions as one option, not the only one.
- **RAG is table stakes.** Embeddings, vector search, chunking. The [Learn RAG course](/docs/courses/ai/rag) covers the JavaScript path.

The [AI Engineer Path](/docs/paths/ai-engineer-path) (around 11.4 hours, Pro) sequences these into one progression. It is the single best reason to pay for Pro in this category. Confirm the live duration and price on the [official path page](https://scrimba.com/home?pricing&via=u42d4986).

## Honest gaps

We like this catalog. We are also clear about what it is **not**:

- **JavaScript and TypeScript only.** No PyTorch, no Hugging Face Transformers in Python, no scikit-learn. If you want classical ML, take Andrew Ng's specialization or fast.ai. Scrimba teaches LLM application engineering, not model training.
- **No ML math.** Linear algebra, calculus, statistics, backpropagation. None of it. Bring it from elsewhere if you need it.
- **No evaluation, RLHF, or fine-tuning depth.** You will learn to call models, not train them.
- **Vendor coverage tilts toward OpenAI, Anthropic, and Mistral.** Less on open-weight self-hosted stacks (Llama, Mistral local, vLLM).
- **LangChain.js abstractions can age fast.** Take the course; do not treat any one framework as required.

That is the honest scope. Within it, the catalog is strong and current.

## Why scrims work for AI specifically

API calls fail in messy ways: a missing header, a stream that closes early, a tool schema the model ignores. Scrimba's format lets you **pause the instructor mid-request and edit the code in place**, so you see exactly where the payload went sideways. That is a better fit than passive video for a topic where every provider's SDK behaves slightly differently.
