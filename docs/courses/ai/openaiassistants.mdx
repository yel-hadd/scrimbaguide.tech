# OpenAI Assistants API on Scrimba

Scrimba has a Pro course on OpenAI's Assistants API, the toolkit for building assistant-style AI apps in JavaScript using threads, file uploads, and tool calling. As of the last catalog review it is a Pro course. This page is a brief orientation to what the Assistants API is and where the course fits on the AI track.

## What it covers

The Assistants API wraps several useful pieces into one product-shaped flow. Threads keep a conversation's history so you do not have to manage state by hand. File uploads let the assistant work with documents you provide. Tool calling lets the assistant trigger your own functions, so it can do things rather than only describe them. Together they make it straightforward to prototype assistant-style apps, including internal copilots, before reaching for heavier custom orchestration.

## Where it fits

This course sits within the [AI Engineer Path](/docs/paths/ai-engineer-path), alongside the other applied AI courses. It assumes you can already write JavaScript and call an API, so it is not a starting point. If you are new to building with AI, begin with the free options first: [Learn to Code with AI](/docs/courses/ai/to-code-with-ai), [Intro to Mistral AI](/docs/courses/ai/mistral-ai), and [Build Serverless AI Agents with Langbase](/docs/courses/ai/serverless-ai-agents-with-langbase). Those will give you the footing to make the Assistants API material land.
