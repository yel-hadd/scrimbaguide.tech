# Scrimba Student Discount

<DisclosureNotice />

**Quick answer:** Three honest paths exist. First, email `help@scrimba.com` from a verified school address and ask for the student rate. Second, claim the [GitHub Student Developer Pack](https://education.github.com/pack) and redeem one free month of Pro on Scrimba's GitHub Education page. Third, write to support if you are in real financial hardship. There is no universal public coupon code; Scrimba grants the discount only after verifying you are actually a student. For current pricing, see the <AffiliateLink href="https://scrimba.com/our-pricing">official Scrimba pricing page</AffiliateLink>.

*Last reviewed: May 2026. Discount terms can change. Confirm with the Scrimba help centre before you act.*

## The three options at a glance

| | GitHub Student Pack | Academic-email discount | Hardship request |
| :--- | :--- | :--- | :--- |
| **What you get** | One free month of Pro | Ongoing reduced Pro rate | Case-by-case goodwill help |
| **Speed** | Fastest (instant once GitHub-approved) | 1 to 3 business days | 1 to 3 business days |
| **Duration** | 30 days, one-time | While you stay verified | Whatever support agrees to |
| **Eligibility** | Verified GitHub Education students | Anyone with a valid academic email | Genuine financial hardship |

Read the rest of this page for the step-by-step on each, then pick the fastest path that applies to you.

## Who this page is for

Students with a `.edu` (or equivalent national academic) email, anyone already approved for GitHub Education, and learners who need a human conversation when price is the blocker. If you are a self-taught learner without a school enrollment, the student paths below do not apply; see the [refund policy](/docs/pricing/refund-policy) and [pricing page](/docs/pricing/) instead.

## Option 1: academic email discount

If you have a valid academic email address, you can ask Scrimba for a student rate.

1. From your school inbox, email `help@scrimba.com`
2. State that you are a student and request the academic discount
3. Scrimba's team verifies the address and applies the discount manually

This is a manual process. There is no automatic checkout coupon. The amount of the discount and how long it lasts is set by Scrimba and can change, so do not assume a number you read on a blog from last year. Ask support directly.

Reference: [Scrimba Help Centre: Do you offer student discounts?](https://scrimba.helpscoutdocs.com/article/19-do-you-offer-student-discounts)

## Option 2: GitHub Student Developer Pack

The [GitHub Student Developer Pack](https://education.github.com/pack) is a free bundle of tools for verified students. It includes one free month of Scrimba Pro. This is often the simplest path if you already have a developer-leaning student profile.

1. Apply for the [GitHub Student Developer Pack](https://education.github.com/pack). You typically need a school email, a photo of your enrollment ID, or both. GitHub does the verification.
2. Once approved, visit the <AffiliateLink href="https://scrimba.com/github-education?via=u42d4986">Scrimba GitHub Education page</AffiliateLink> and sign in with GitHub.
3. In the checkout modal look for the **GitHub Student 30-Day Trial** message and the yellow **UPGRADE** button. Click it to activate your free Pro month.

Reference: [Scrimba Help Centre: How do I get the Student GitHub free access?](https://scrimba.helpscoutdocs.com/article/46-how-do-i-access-the-student-github-free-access)

The free month is one shot, not recurring. After it ends you fall back to free unless you subscribe. If you also qualify for the academic-email discount, stack them in sequence: use the GitHub free month first, then subscribe at the student rate.

## Option 3: financial hardship

Scrimba does not run a formal scholarship programme, and you should be skeptical of any third-party site that says otherwise. What they do say in their help centre is that you can write to `help@scrimba.com` if you are in genuine hardship. Be honest, explain the situation, and accept that any help is a goodwill exception, not an entitlement.

Reference: [Scrimba Help Centre: Can I get a scholarship?](https://scrimba.helpscoutdocs.com/article/71-can-i-get-a-scholarship)

## Which option to pick

- **Use the GitHub Student Pack** if you want to try Pro free for a month before deciding. This is the fastest path and you keep the rest of the GitHub Pack benefits regardless.
- **Use the academic-email discount** if you want an ongoing reduced rate while you study.
- **Write the hardship email** if neither of the above applies but you are blocked on price. Do not lead with this if you actually qualify for the others.

## How to actually claim this, this week

Pick the fastest path that applies to you and do it now. If you already have GitHub Student approval, log in to the <AffiliateLink href="https://scrimba.com/github-education">Scrimba GitHub Education page</AffiliateLink> and click the upgrade button; the free Pro month starts immediately. If you only have a school email, draft a short message to `help@scrimba.com` from that inbox today, then plan for a 1 to 3 business day reply before paying anything. If neither applies, start on the free tier while you decide; the <AffiliateLink href="https://scrimba.com/s0v687325e">demo scrim</AffiliateLink> takes under a minute and there is no signup.

## What you actually get as a student subscriber

The student rate buys the same Pro account as any other subscriber:

- Full course catalog and all four career paths
- Unlimited interactive challenges
- Private Discord community access
- Path Certificates of Completion (read the [accreditation FAQ](/docs/how-it-works/accreditation) before you assume what these mean to employers)

## When seasonal promotions tend to appear

Beyond the student paths, Scrimba runs occasional platform-wide promotions. The pattern over recent years has been the usual SaaS calendar: a Black Friday and Cyber Monday window in late November, and an occasional New Year push in early January. These are time-boxed and the size varies year to year, so do not bank on a specific number. We track the live picture in the [2026 Scrimba discount codes guide](/blog/scrimba-discount-codes-2026); the durable savings for students remain the GitHub Pack month and the academic-email rate, not seasonal coupons. As always, the <AffiliateLink href="https://scrimba.com/our-pricing">official pricing page</AffiliateLink> is the only authority on what is live right now.

## Honest caveats

- No public coupon code works. If you see "SCRIMBA50" or similar on a coupon site, treat it as fiction unless it is on Scrimba's own site.
- Discounts apply after verification, not at checkout. Plan for a 1 to 3 business day turnaround on the email path.
- Pricing changes. Anything specific you read here about percentage off can drift. Always confirm with support directly before you assume a number.
