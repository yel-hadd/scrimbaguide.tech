# Practice Coding on Scrimba

**Quick answer.** The fastest way out of tutorial hell is to build, not watch. These pages point you to interactive Scrimba drills and projects where you type code in the browser, grouped by skill. Start with the fundamentals you are shakiest on, then move to the project pages once the drills feel easy. Everything here is hands-on; passive reading is the thing we are trying to replace.

## Practice by skill

**CSS and layout**
- **[Practice CSS Grid](/docs/practice/practice-css-grid)**: build real grids instead of memorising properties.
- **[Practice CSS Flexbox](/docs/practice/practice-flexbox)**: the layout model you will reach for daily.
- **[Practice Tailwind CSS](/docs/practice/practice-tailwind-css)**: utility-first styling, hands-on.

**JavaScript and TypeScript**
- **[Practice JavaScript Arrays](/docs/practice/practice-javascript-arrays)**: map, filter, reduce until they are reflex.
- **[Practice API Calls](/docs/practice/practice-api-calls)**: fetch, async/await, and handling real responses.
- **[Practice TypeScript](/docs/practice/practice-typescript)**: types and interfaces on code you actually write.

**React**
- **[Practice React Hooks](/docs/practice/practice-react-hooks)**: useState, useEffect, and the rules of hooks.
- **[React Practice Projects](/docs/practice/practice-react-projects)**: small builds that stretch you past tutorials.
- **[Build a Weather App in React](/docs/practice/build-a-weather-app-interactively)**: a guided end-to-end build.
- **[React Portfolio Project Ideas](/docs/practice/react-portfolio-project-ideas)**: prompts worth putting in front of employers.

**AI engineering**
- **[Practice AI Engineering](/docs/practice/practice-ai-engineering)**: drills for agents, RAG, and API-driven AI features.

## Where practice fits in the bigger picture

Drills are the reps; the [Learning Paths](/docs/paths/) are the curriculum. If you are following a path, use these pages to reinforce a weak spot before moving on. If you are deciding whether Scrimba's interactive format works for you at all, [how scrims work](/docs/how-it-works/how-scrims-work) explains the pause-and-code loop, and [escaping tutorial hell](/docs/how-it-works/tutorial-hell) covers why building beats watching.
