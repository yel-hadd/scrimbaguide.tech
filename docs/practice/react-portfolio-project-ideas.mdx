## Who this is for

Career switchers and juniors who need **three strong repos**, not ten half-finished tutorials.


## 1. The Movie Search Database

A staple of the **Advanced React** curriculum. This project proves you can work with large external datasets.

**What you will learn:**
- Interfacing with the OMDB or TMDB API.
- Debouncing search inputs to reduce API load.
- Complex grid layouts using CSS.
- Handling empty states (when a search returns no results).

**Why it impresses employers:** It shows you know how to build a search interface, which is a core feature of almost every B2B software product.

## 2. Type-Safe E-Commerce Cart

Building a shopping cart with React and TypeScript shows you understand data validation and complex state updates.

**What you will learn:**
- Using `useReducer` or React Context to manage the cart globally.
- Updating quantities and recalculating totals dynamically.
- Defining strict TypeScript interfaces for products and users.

**Why it impresses employers:** Working with money/carts implies you write robust, bug-free code. TypeScript is highly demanded.

## 3. Real-Time Chat App with Firebase

A step up in complexity that demonstrates back-end integration skills.

**What you will learn:**
- Integrating React with a BaaS (Backend-as-a-Service) like Firebase or Supabase.
- Using `useEffect` to subscribe to database channels.
- Auth (Logging users in and out safely).

**Why it impresses employers:** Most frontends are stateless. Showing you can manage a persistent database connection sets you apart from entry-level applicants.
