# What's New in Scrimba?

**Quick Answer:** We track Scrimba platform updates, new courses, and features. Recent: AI Instant Feedback, React 19 updates, Bootcamp Git collab, Backend Path launch.

*Last reviewed: March 2026.*

We track major updates to the Scrimba platform so you know what's new. Sources: [Scrimba roadmap](https://scrimba.com/roadmap?via=u42d4986), [Scrimba blog](https://scrimba.com/blog?via=u42d4986), and Discord community announcements.

## February 2026

- **New Feature**: AI-powered Instant Feedback on challenges — get hints when you're stuck.
- **Course Update**: Learn React updated with React 19 patterns (Server Components, Actions, use hook).

## January 2026

- **New Feature**: Bootcamp group projects now support real-time Git collaboration.
- **New Course**: Advanced Next.js 15 added to the Fullstack Path.

## Late 2025

- **Launch**: [Backend Developer Path](/docs/paths/backend-developer-path) officially launched out of beta.

## Mid 2025

- **Update**: React 19 course update rolled out to all users.
- **Feature**: Scrimba 2.0 editor performance improvements.

## Ongoing

- Regular "Code Review of the Week" videos.
