# What’s new on Scrimba?

We summarize platform and catalog changes so you are not guessing from scattered tweets. Recent highlights:

- AI Instant Feedback on challenges
- React 19 course coverage
- Bootcamp Git and GitHub workflows
- Backend Developer Path out of beta

_Last reviewed: July 2026._

**Why read this?** If you are deciding whether Pro is “stale,” seeing recent ship velocity matters. We pull from the public Scrimba roadmap, blog, and community announcements, then point you to the [learning paths](/docs/paths/) when a drop affects what you should study next.

## Early 2026

- **Course update**: The "What's New in React 19" course covers the newest React 19 patterns (Server Components, Actions, the `use` hook). These live in that dedicated course rather than in the foundational Learn React course.
- **Feature**: Bootcamp group projects use Git and GitHub (branching, pull requests, code review), so you practice a real team workflow.

## Late 2025

- **Launch**: [Backend Developer Path](/docs/paths/backend-developer-path/) officially launched out of beta.
- **New Courses**:
  - Context Engineering for AI
  - Building with Vercel AI SDK
  - Learn SQL

## Early 2025

- **New Feature**: AI-powered Instant Feedback on challenges launched in February 2025, giving you hints when you're stuck.

## 2024

- **Feature**: The Scrimba 2.0 editor shipped in July 2024, with major performance improvements over the original platform.

## Ongoing

- Weekly Town Halls in Discord.
- Regular "Code Review of the Week" videos.
