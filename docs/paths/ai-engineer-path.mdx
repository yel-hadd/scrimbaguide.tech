## What You'll Learn

The AI Engineer Path is Scrimba's newest and fastest-growing path. It teaches web developers how to **build AI-powered applications** using modern tools and frameworks.

## Complete Module Breakdown

| Module | Duration | Lessons | Topics |
|---|---|---|---|
| Intro to AI Engineering | 2.4 hrs | 20 | LLM basics, API calls, prompt design |
| Deployment | 52 min | 1 | Deploying AI apps (Cloudflare) |
| Embeddings and Vector Databases | 95 min | 3 | Vector search, semantic similarity |
| Agents | 117 min | 6 | Autonomous AI agents, tool use |
| Context Engineering | 58 min | 5 | Optimizing LLM context windows |
| Vercel AI SDK | 113 min | 6 | Building AI UIs with Vercel |
| Model Context Protocol | 37 min | 3 | MCP servers and clients |
| Multimodality | 62 min | 2 | Image, audio, and video AI |

## Who This Is For

- **Web developers** who want to add AI skills to their toolkit
- **JavaScript/TypeScript developers** — the path uses JS throughout
- Anyone building **AI-powered products** (startups, agencies, enterprise)
- Developers curious about **agents, RAG, and MCP**

## Cutting-Edge Topics

This path covers the most in-demand AI engineering topics of 2025-2026:

- **AI Agents** — autonomous systems that use tools and make decisions
- **RAG (Retrieval-Augmented Generation)** — combining LLMs with your own data
- **Model Context Protocol (MCP)** — the emerging standard for AI tool integration
- **Context Engineering** — optimizing what you send to LLMs
- **Vercel AI SDK** — building production AI user interfaces
