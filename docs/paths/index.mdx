# Which Scrimba career path fits your goal?

**Quick Answer:** Scrimba has 4 career paths — Frontend (81.6h), Fullstack (108.4h), Backend (30.1h), and AI Engineer (11.4h). All require Pro. Choose Frontend if you're a beginner; Fullstack for the most comprehensive training; Backend if you know frontend basics; AI Engineer to add AI skills to your toolkit.

*Last reviewed: March 2026. Path durations and content may change.*

## Start here if you're choosing a path

You're comparing Frontend vs Fullstack vs Backend vs AI—not just reading feature lists. The sections below map each path to a typical learner (beginner, career changer, working dev adding skills) so you can align hours and outcomes with what you actually need next.

## Which Path Should You Choose?

**Choose [Frontend](/docs/paths/frontend-developer-path) if:**
- You're a complete beginner
- You want to build websites and web apps
- You're aiming for a frontend developer role
- Created in partnership with **Mozilla MDN**

**Choose [Fullstack](/docs/paths/fullstack-developer-path) if:**
- You want the most comprehensive training
- You want to learn both frontend and backend
- You want to include AI engineering skills
- You have 100+ hours to commit

**Choose [Backend](/docs/paths/backend-developer-path) if:**
- You already know some frontend basics
- You want to focus on servers, databases, and APIs
- You're interested in DevOps and cybersecurity

**Choose [AI Engineer](/docs/paths/ai-engineer-path) if:**
- You already have web development experience
- You want to add AI/ML skills to your toolkit
- You're interested in building AI-powered apps
