## Who Is This Path For?

- Developers who **already know frontend basics** and want to add backend skills
- Anyone interested in **server-side JavaScript** (Node.js ecosystem)
- Those targeting **backend or full-stack roles**
- The path is marked **Intermediate** — some prior programming experience is recommended

## Backend vs Fullstack Path

The [Fullstack Path](/docs/paths/fullstack-developer-path) (108.4 hrs) includes all frontend content plus backend. The Backend Path (30.1 hrs) is focused and faster — ideal if you already know frontend.
