## What You'll Learn

The Fullstack Developer Path is Scrimba's **most comprehensive offering** — over 108 hours covering everything from HTML basics to Next.js and AI engineering.

## Complete Module Breakdown

| Module | Topics Covered |
|---|---|
| Introduction | Path overview and setup |
| HTML and CSS Fundamentals | Semantic HTML, CSS basics, layouts |
| JavaScript Fundamentals | Variables, functions, DOM manipulation |
| Tools of the Trade | Git, command line, Vite |
| Accessible Development | ARIA, keyboard nav, screen readers |
| Essential CSS | Grid, Flexbox, animations, variables |
| Essential JavaScript | Closures, promises, modules, ES6+ |
| Responsive Design | Media queries, mobile-first, fluid layouts |
| APIs and Async JavaScript | Fetch, REST, async/await |
| AI Engineering | LLMs, agents, RAG fundamentals |
| Node.js | Server-side JavaScript, modules |
| Databases | SQL, Supabase, data modeling |
| Express.js | Routing, middleware, REST APIs |
| User Interface Design | UI/UX principles |
| React.js Fundamentals | Components, state, hooks |
| Testing | Unit testing, test-driven development |
| Advanced React.js | Context, routing, patterns |
| TypeScript | Types, interfaces, generics |
| Next.js | SSR, routing, full-stack React |
| Launching Your Career | Portfolio, interviews, job search |

**Total: 108.4 hours across 94 modules**

## Who This Is For

- **Ambitious beginners** who want to learn everything
- **Career changers** targeting full-stack developer roles
- Developers who want **one path instead of multiple**
- Anyone interested in covering both **frontend, backend, and AI**

## Fullstack vs Frontend Path

| | Fullstack | Frontend |
|---|---|---|
| Duration | 108.4 hrs | 81.6 hrs |
| Backend (Node/Express) | Yes | No |
| Databases (SQL) | Yes | No |
| AI Engineering | Yes | No |
| TypeScript | Yes | No |
| Next.js | Yes | No |
| Testing | Yes | No |

The Fullstack Path includes everything in the Frontend Path **plus** backend, databases, TypeScript, testing, AI, and Next.js.

Read more: [Frontend vs Fullstack: Which to Choose?](/blog/scrimba-frontend-vs-fullstack-path)

## Choose This If

**Choose this path if:** You want the most comprehensive training and plan to commit 100+ hours. Includes frontend, backend, databases, TypeScript, and AI. Choose Frontend if you only want web dev; Backend if you already know frontend.
