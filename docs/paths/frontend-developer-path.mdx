## What You'll Learn

The Frontend Developer Path takes you from absolute zero to a job-ready frontend developer. It's Scrimba's most popular career path and was created **in partnership with Mozilla MDN**.

## Complete Module Breakdown

| Module | Duration | Lessons | What You'll Learn |
|---|---|---|---|
| Welcome to the career path | 26 min | 7 | Orientation and expectations |
| Web dev basics | 6.3 hrs | 80 | HTML & CSS foundations |
| Making websites interactive | 9.6 hrs | 144 | JavaScript fundamentals |
| Accessible development | 95 min | 23 | Web accessibility (a11y) |
| Essential CSS concepts | 4.5 hrs | 50 | Grid, Flexbox, variables |
| Essential JavaScript concepts | 10.6 hrs | 121 | Advanced JS patterns |
| Responsive design | 4.6 hrs | 54 | Mobile-first, media queries |
| Code reviews | 22 min | 4 | Review practices |
| Working with APIs | 7.6 hrs | 96 | Fetch, async/await, REST |
| User interface design | 2.5 hrs | 13 | UI/UX principles |
| React basics | 15.3 hrs | 157 | React fundamentals |
| Advanced React | 13.3 hrs | 125 | Hooks, routing, patterns |
| Getting hired | 5.1 hrs | 73 | Portfolio, interviews, job search |

**Total: 81.6 hours across 947+ lessons**

## Who Is This Path For?

- **Complete beginners** with no coding experience
- **Career changers** looking to break into tech
- **Self-taught developers** wanting structured learning
- Anyone who prefers **interactive, hands-on** learning over passive videos

## Pros

- Created with **Mozilla MDN** — industry-standard curriculum
- Covers the **full stack** of frontend skills (HTML → CSS → JS → React)
- Includes **career prep** module (portfolio, interviews, job search)
- Interactive format means you **code from day one**
- Certificate of completion for LinkedIn

## Cons

- Requires **Scrimba Pro** subscription ([pricing](/docs/pricing/))
- 81.6 hours is a significant time commitment
- Backend skills not covered (see [Fullstack Path](/docs/paths/fullstack-developer-path))
