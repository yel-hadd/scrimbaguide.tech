# Scrimba pricing summary

Source: https://scrimbaguide.tech/docs/pricing/
Last reviewed: 2026.

## Plans

- **Free** — $0. About 19 full courses (Learn React, Learn JavaScript, Learn TypeScript and more), 10 total challenges. No Discord, no path certificates. Free tier never expires; no credit card required.
- **Pro Monthly** — Per-month billing. Unlocks all 86+ courses, 4 career paths, unlimited challenges, private Discord, and certificates of completion.
- **Pro Annual** — Annual billing, normally the best effective per-month rate. Same feature set as Pro Monthly.

Prices change — confirm the current number on Scrimba's pricing page: https://scrimba.com/home?pricing

## Career paths included with Pro

- Backend Developer Path — 30.1 hours.
- AI Engineer Path — 11.4 hours (agents, RAG, MCP).
